## 🚀 Getting Started

1. **Prerequisites**
   ```bash
   node.js >= 18.17
   npm or yarn or pnpm
   ```

2. **Installation**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Development**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Build**
   ```bash
   npm run build
   # or
   yarn build
   ```
